# Minerals

A site for viewing various gems.

## Getting Started

Download the files.

### Prerequisites

```
PIP
Python3
Django
```

### Installing

Clone the repo
Install Django
Run the migrations

```
git clone https://github.com/jawanng/minerals.git
cd minerals
pip install -r requirements.txt 
python manage.py migrate
```

## Deployment

```
python manage.py runserver
```
